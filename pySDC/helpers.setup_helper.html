helpers.setup_helper module¶

generate_description(problem_class, **kwargs)[source]¶

Generate a description object that you can use to run pySDC based on a problem class and various input. This function does not set any additional defaults, but distributes the values to where they belong.

Parameters:

problem_class (pySDC.Problem) – A problem class

Returns:

A description object for running pySDC

Return type:

dict
