class testequation0d(lambdas=None, u0=0.0)[source]¶

Bases: ptype

This class implements the simple test equation of the form

\[\frac{d u(t)}{dt} = A u(t)\]

for \(A = diag(\lambda_1, .. ,\lambda_n)\).

Parameters:

  • lambdas (sequence of array_like) – List of lambda parameters.

  • u0 (sequence of array_like) – Initial condition.

A¶

Diagonal matrix containing \(\lambda_1,..,\lambda_n\).

Type:
